    AI와의 첫만남 — 인공지능, 넌 누구니?¶

    Part 1 | [1권] AI와 만나기 — 인공지능의 세계로 첫 발걸음

    🎯 이 장에서 배우는 것¶

    이 장을 마치면 다음을 할 수 있어요.

    • 인공지능(AI)의 개념을 자기만의 말로 설명할 수 있다
    • 일상 속에서 AI가 활용되는 사례를 5가지 이상 찾을 수 있다
    • AI가 잘하는 것과 못하는 것을 구분하고 비판적으로 판단할 수 있다
    • AI 기술의 발전 과정을 시대별로 정리할 수 있다

    💡 왜 이걸 배우나요?¶

    아침에 눈을 떠서 잠들 때까지, 우리는 이미 AI와 함께 살고 있습니다.

    스마트폰이 추천해주는 영상, 자동으로 분류되는 사진 앨범, "오늘 비 와?" 하고 물어보는 음성 비서, 시험 기간에 몰래 써보는 번역기까지. 이 모든 것의 뒤에는 인공지능(AI) 이라는 기술이 숨어 있습니다.

    그런데 정작 "AI가 뭐야?"라고 물으면, 명확하게 대답할 수 있는 사람은 많지 않습니다.

    🤔 생각해보기: 오늘 하루 동안 나는 AI를 몇 번이나 만났을까요? 아침에 일어나서 지금 이 교재를 펼치기까지의 과정을 떠올려보세요.

    이 장에서는 AI의 정체를 밝히고, AI가 정말 '똑똑한' 존재인지 직접 실험해볼 거예요. 미래를 살아갈 여러분이 AI를 제대로 이해하고, 현명하게 활용하는 첫걸음을 내딛는 시간입니다.

    📚 핵심 개념¶

    개념 1: 인공지능(AI)이란 무엇인가?¶

    🎈 비유로 시작하기¶

    여러분이 강아지를 키운다고 상상해보세요.

    처음에 강아지는 아무것도 모릅니다. 하지만 "앉아!"라고 말하고, 앉을 때마다 간식을 주면 어떻게 될까요? 반복하다 보면 강아지는 "앉아"라는 소리를 듣고 앉는 법을 학습합니다.

    인공지능도 비슷합니다. 컴퓨터에게 수많은 데이터(경험)를 주고, 규칙(간식)을 알려주면, 컴퓨터가 스스로 판단하는 법을 배웁니다. 다만 강아지와 다른 점이 하나 있어요. 컴퓨터는 수백만 장의 사진을 단 몇 시간 만에 학습할 수 있다는 것이죠!

    📖 정확한 정의¶

    인공지능(Artificial Intelligence, AI) 이란, 인간의 학습 능력, 판단 능력, 언어 이해 능력 등을 컴퓨터 프로그램으로 구현한 기술입니다.

    쉽게 말하면, "사람처럼 생각하고 판단할 수 있도록 만든 컴퓨터 프로그램" 입니다.

    여기서 핵심 단어를 짚어볼까요?

    • 인공(Artificial): 자연적으로 생긴 것이 아니라 사람이 만든 것
    • 지능(Intelligence): 배우고, 이해하고, 판단하는 능력

    즉, AI는 자연이 만든 지능(사람의 뇌)을 본떠서 사람이 인공적으로 만든 지능입니다.

    ✅ 예시로 확인하기¶

    다음 중 AI가 하는 일은 무엇일까요?

    • 🟢 AI — 유튜브가 내 취향에 맞는 영상을 추천해준다
    • 🟢 AI — 네이버 파파고가 영어를 한국어로 번역해준다
    • 🔴 AI 아님 — 계산기가 1+1=2를 계산한다
    • 🔴 AI 아님 — 선풍기 타이머가 3시간 뒤에 꺼진다

    💡 알아보기: 계산기나 타이머는 사람이 미리 정해놓은 규칙 그대로 작동할 뿐이에요. AI는 데이터를 보고 스스로 패턴을 찾아내서 판단한다는 점이 다릅니다.

    개념 2: 일상 속의 AI — 이미 우리 곁에!¶

    AI는 먼 미래의 기술이 아닙니다. 지금 이 순간에도 여러분의 주머니 속 스마트폰에서 작동하고 있어요.

    🔍 하루 속 AI 찾기¶

    아침부터 밤까지, AI가 숨어 있는 순간들을 따라가 봅시다.

    🌅 아침 - 스마트폰 얼굴 인식 잠금 해제 → AI가 내 얼굴의 특징점을 분석 - 날씨 앱의 "오후에 비 올 확률 80%" → AI가 기상 데이터를 분석해서 예측

    🏫 학교 가는 길 - 네비게이션의 최적 경로 추천 → AI가 실시간 교통 데이터를 분석 - 스팸 문자 자동 차단 → AI가 문자 내용을 읽고 스팸인지 판단

    📱 쉬는 시간 - 유튜브 추천 알고리즘 → 내가 본 영상 패턴을 분석해서 다음 영상 추천 - 인스타그램 필터 → AI가 얼굴 위치를 인식해서 효과 적용 - 번역 앱으로 영어 해석 → AI가 문맥을 이해하고 자연스러운 번역 생성

    🌙 집에서 - "시리야, 알람 맞춰줘" → AI가 음성을 텍스트로 변환하고 의미를 해석 - 넷플릭스 "당신이 좋아할 만한 영화" → 시청 이력을 분석한 AI 추천

    개념 3: AI가 잘하는 것 vs 못하는 것¶

    영화 속에서 AI는 만능처럼 보입니다. 하지만 현실의 AI에는 뚜렷한 강점과 한계가 있습니다.

    💪 AI가 잘하는 것¶

    1. 대량의 데이터 분석 사람이 100만 장의 엑스레이 사진을 보려면 몇 년이 걸리지만, AI는 단 며칠이면 분석을 끝냅니다.

    2. 반복적인 패턴 인식 고양이 사진 100만 장을 학습한 AI는 새로운 고양이 사진도 높은 정확도로 알아봅니다.

    3. 24시간 쉬지 않고 일하기 AI 챗봇은 새벽 3시에도, 명절에도, 쉬지 않고 응대합니다.

    4. 빠른 속도의 계산과 예측 바둑 AI '알파고'는 1초에 수만 가지 수를 계산합니다.

    😅 AI가 못하는 것 (아직은!)¶

    1. 진짜로 '이해'하기 AI 번역기는 문장을 번역할 수 있지만, 그 문장이 슬픈 이야기인지 기쁜 이야기인지 진심으로 느끼지는 못합니다.

    2. 상식적 판단 "비가 오면 우산을 가져가야 해"는 사람에게 당연하지만, AI에게는 별도로 학습시켜야 하는 지식입니다.

    3. 창의적이고 새로운 문제 해결 AI는 기존 데이터에서 패턴을 찾는 것은 잘하지만, 한 번도 본 적 없는 완전히 새로운 상황에서는 엉뚱한 답을 내놓기도 합니다.

    4. 윤리적 판단 자율주행 차가 사고 상황에서 "누구를 보호할 것인가?"와 같은 도덕적 결정을 내리는 것은 AI에게 매우 어려운 문제입니다.

    💡 알아보기: 최근 ChatGPT 같은 생성형 AI가 등장하면서 "AI도 창작할 수 있는 거 아니야?"라는 질문이 많아요. 생성형 AI는 기존 데이터의 패턴을 조합해서 새로운 결과물을 만들어내는 것이지, 사람처럼 영감을 받아 창작하는 것과는 다릅니다. 이 차이를 기억해두세요!

    개념 4: AI의 역사 — 꿈에서 현실로¶

    AI는 갑자기 나타난 기술이 아닙니다. 약 70년에 걸친 긴 여정이 있었어요.

    ⏳ AI 역사를 4단계로 정리¶

    🌱 1단계: 탄생기 (1950년대)

    1950년, 영국의 수학자 앨런 튜링이 "기계도 생각할 수 있을까?"라는 질문을 던졌습니다. 이것이 AI 연구의 시작이에요. 1956년에는 미국 다트머스 회의에서 "인공지능(Artificial Intelligence)" 이라는 이름이 처음 붙여졌습니다.

    앨런 튜링이 제안한 튜링 테스트: 사람이 컴퓨터와 대화했을 때, 상대가 사람인지 컴퓨터인지 구분할 수 없다면 그 컴퓨터는 '지능이 있다'고 볼 수 있다는 테스트입니다.

    ❄️ 2단계: AI 겨울 (1970~1990년대)

    초기의 기대와 달리, 당시 컴퓨터는 너무 느리고 데이터도 부족했습니다. "AI는 실패한 꿈"이라는 비판이 쏟아졌고, 연구 지원금이 줄면서 AI 연구가 크게 위축되었어요. 이 시기를 'AI 겨울' 이라고 부릅니다.

    🔥 3단계: 부활기 (2000~2010년대)

    인터넷이 발전하면서 어마어마한 양의 데이터가 쌓이기 시작했고, 컴퓨터의 계산 능력도 폭발적으로 성장했습니다. 특히 딥러닝(Deep Learning) 이라는 새로운 학습 방법이 등장하면서 AI의 성능이 급격히 향상되었어요.

    • 2011년 — IBM의 AI '왓슨'이 퀴즈쇼에서 인간 챔피언을 이김
    • 2016년 — 구글의 AI '알파고'가 바둑 세계 챔피언 이세돌 9단을 이김 🇰🇷

    🚀 4단계: 폭발적 성장기 (2020년대~현재)

    ChatGPT, DALL-E, Midjourney 같은 생성형 AI가 등장하면서, AI는 글을 쓰고, 그림을 그리고, 코드를 작성하는 수준에 이르렀습니다. 이제 AI는 전문가만의 도구가 아니라 누구나 매일 사용하는 기술이 되었습니다.

    timeline title AI의 역사 — 꿈에서 현실로 1950년대 : 🌱 탄생기 : 튜링 테스트 제안 : "인공지능" 용어 탄생 1970~90년대 : ❄️ AI 겨울 : 기술적 한계 : 연구 지원 감소 2000~2010년대 : 🔥 부활기 : 빅데이터 + 딥러닝 : 알파고 vs 이세돌 2020년대~ : 🚀 폭발적 성장 : ChatGPT 등 생성형 AI : 일상 속 AI 시대

    🤔 생각해보기: AI 겨울은 왜 왔을까요? 기술이 아무리 좋아도 그것을 뒷받침할 컴퓨터 성능과 데이터가 부족하면 발전할 수 없다는 교훈을 줍니다. 지금의 AI가 이렇게 발전할 수 있었던 이유도 스마트폰, 인터넷, 클라우드 등 다른 기술의 발전 덕분이에요.

    🔨 따라하기: "멍청한 AI 찾기" 실험¶

    AI가 정말 똑똑한지, 직접 실험해봅시다! ChatGPT나 다른 AI 챗봇에게 엉뚱한 질문을 던져서 AI의 한계를 체험하는 활동입니다.

    Step 1: AI 챗봇 준비하기¶

    다음 중 하나를 선택해서 접속하세요 (무료로 사용 가능).

    • ChatGPT: chat.openai.com
    • 뤼튼: wrtn.ai
    • Bing Chat: bing.com/chat

    ⚠️ 회원가입이 필요할 수 있어요. 학교에서 진행한다면 선생님의 안내를 따르세요.

    Step 2: 기본 질문 던져보기¶

    먼저 AI가 잘 대답하는 질문부터 시작합니다.

    질문 예시:
"대한민국의 수도는 어디야?"
"물의 화학식은 뭐야?"
"1부터 100까지 더하면 얼마야?"

    예상 결과: AI가 정확하게 대답할 거예요. ✅

    Step 3: 엉뚱한 질문으로 AI 당황시키기 🤪¶

    이제 AI가 어려워할 질문을 던져봅시다!

    엉뚱한 질문 예시:

1. "코끼리를 냉장고에 넣으려면 몇 단계가 필요해?"
   → AI가 진지하게 대답하는지 관찰

2. "내가 지금 슬픈데, 진짜로 내 마음을 이해해?"
   → AI의 '공감'이 진짜인지 생각해보기

3. "이 문장은 거짓말이야. 이 말이 맞아, 틀려?"
   → 논리적 모순(패러독스)을 어떻게 처리하는지 관찰

4. "어제 우리 반에서 있었던 일 알아?"
   → AI가 모르는 정보(개인적 경험)에 어떻게 반응하는지 확인

    Step 5: 실험 결과 분석하기¶

    실험을 마쳤으면, 다음 질문에 답해보세요.

    1) AI가 가장 잘 대답한 질문은? → 왜 잘 대답했을까요? (힌트: 인터넷에 정보가 많은 질문)

    2) AI가 가장 엉뚱한 대답을 한 질문은? → 왜 못 대답했을까요? (힌트: 상식, 감정, 개인 경험)

    3) AI는 '모른다'고 솔직하게 말했나요, 아니면 모르면서도 그럴듯한 답을 지어냈나요? → 이것이 AI의 가장 위험한 특성 중 하나입니다!

    ⚠️ 중요: AI가 그럴듯하게 지어낸 거짓 정보를 '할루시네이션(Hallucination, 환각)' 이라고 합니다. AI가 말하는 것을 무조건 믿으면 안 되는 이유가 바로 여기에 있어요!

    📝 핵심 정리 코드¶

    AI 개념을 Python 코드로 체험해봅시다. 아래 코드는 아주 간단한 "규칙 기반 AI"와 "학습 기반 AI"의 차이를 보여줍니다.

    # ========================================
# 🤖 나의 첫 번째 AI 체험 코드
# 규칙 기반 vs 학습 기반의 차이 이해하기
# ========================================

# --- 1단계: 규칙 기반 프로그램 (AI 아님!) ---
def calculator(a, b, operation):
    """사람이 정해준 규칙대로만 작동하는 계산기"""
    if operation == "더하기":
        return a + b
    elif operation == "빼기":
        return a - b
    else:
        return "모르는 연산입니다"

print("=== 규칙 기반 계산기 ===")
print(f"3 더하기 5 = {calculator(3, 5, '더하기')}")
print(f"10 빼기 4 = {calculator(10, 4, '빼기')}")
print(f"3 곱하기 5 = {calculator(3, 5, '곱하기')}")
# → '곱하기'는 규칙에 없어서 모른다고 답함!

print()

# --- 2단계: 학습 기반 프로그램 (간단한 AI!) ---
def simple_ai_classifier(text):
    """
    간단한 감정 분석 AI
    긍정적/부정적 단어 목록을 '학습 데이터'로 사용
    """
    # AI가 학습한 데이터 (실제 AI는 수백만 개의 데이터로 학습!)
    positive_words = ["좋아", "행복", "최고", "재미있", "사랑", "멋져", "훌륭"]
    negative_words = ["싫어", "슬퍼", "최악", "지루", "나빠", "힘들", "짜증"]

    positive_count = 0
    negative_count = 0

    for word in positive_words:
        if word in text:
            positive_count += 1

    for word in negative_words:
        if word in text:
            negative_count += 1

    # AI의 판단
    if positive_count > negative_count:
        return "😊 긍정적인 문장이네요!"
    elif negative_count > positive_count:
        return "😢 부정적인 문장이네요..."
    else:
        return "🤔 잘 모르겠어요... 더 많은 학습이 필요해요!"

print("=== 간단한 감정 분석 AI ===")

test_sentences = [
    "오늘 날씨가 좋아서 행복해!",
    "시험이 너무 어려워서 슬퍼...",
    "오늘 점심 뭐 먹지?",
    "최악의 하루였지만 친구 덕분에 좋아졌어"
]

for sentence in test_sentences:
    result = simple_ai_classifier(sentence)
    print(f"입력: '{sentence}'")
    print(f"AI 판단: {result}")
    print()

AI 판단: 🤔 잘 모르겠어요... 더 많은 학습이 필요해요!

    💡 코드에서 배우는 핵심: - 규칙 기반 프로그램은 사람이 정해준 것만 할 수 있어요 (곱하기를 안 알려주면 모름). - 학습 기반 AI는 데이터(긍정/부정 단어)를 보고 스스로 판단해요. - 하지만 마지막 문장처럼 "최악"과 "좋아"가 섞인 복잡한 문장은 아직 제대로 판단하지 못해요. 더 많은 데이터와 더 정교한 알고리즘이 필요합니다!

    ⚠️ 자주 하는 실수¶

    실수 1: "AI는 스스로 생각한다" ❌¶

    AI는 생각하는 것이 아니라 패턴을 계산하는 것입니다. ChatGPT가 사람처럼 말한다고 해서 진짜 '생각'하고 있는 건 아니에요. 수학적으로 "이 단어 다음에 올 확률이 가장 높은 단어"를 계산해서 문장을 만드는 것입니다.

    🔑 기억하세요: AI는 "이해"하는 것이 아니라 "계산"하는 것!
